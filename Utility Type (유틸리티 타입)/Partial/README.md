### Partial 타입
- 객체의 필수 타입까지 전부 선택적 타입으로 변경하고 싶을때 사용
- 객체의 모든 타입이 선택적 타입으로 변경되었기 때문에 값이 없어도 에러가 발생하지 않음

    ~~~
    Partial<객체 타입>
  
  
    EX-1)
    interface 인터페이스명 {
        타입명1: 타입;
        타입명2?: 타입;
    }
  
    const 변수명: Partial<인터페이스명> = {}
  
  
    EX-2)
    type 타입이름1 = {
        타입명1: 타입;
        타입명2?: 타입; 
    }
  
    type 타입이름2 = Partial<타입이름1>
  
    const 변수명: 타입이름2 = {}
    ~~~
